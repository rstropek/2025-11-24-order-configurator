## Product Master Data Management

### Background

This application manages product master data including categories, products, and their attributes. It provides functionalities for creating, updating, and deleting product information, as well as organizing products into categories for better management and retrieval.

### Technologies

* TypeScript
* Node.js
* SQLite
* Drizzle ORM
* ZOD for schema validation

### Coding Guidelines

* Prefer `type` over `interface` for defining data structures.
* Use ZOD schemas for data validation and transformation.
